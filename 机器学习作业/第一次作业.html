2 实验原理

   Robbins-Monro算法由Herbert Robbins和Sutton Monro于1951年提出，提供了一种寻找未知回归函数的零点或极值的递推方法。假设有一个函数M(θ)M(\theta)M(θ)和一个常量α\alphaα，方程M(θ)=αM(\theta)=\alphaM(θ)=α有唯一的根θ∗\theta^*θ∗。我们无法直接地观察到M(θ)M(\theta)M(θ)，但可以得到随机变量N(θ)N(\theta)N(θ)的观察值从而进行求解，其中E[N(θ)]=M(θ)E[N(\theta)]=M(\theta)E[N(θ)]=M(θ)。该算法可以用如下的迭代方程表示：
θn+1=θn−an(N(θn)−α)(1)\theta_{n+1}=\theta_n-a_n(N(\theta_n)-\alpha) \tag{1}θn+1​=θn​−an​(N(θn​)−α)(1)
其中，{an}\{a_n\}{an​}是一个递增序列且满足下列条件：
lim⁡N→+∞aN=0(2)\lim_{N\rightarrow+\infty}a_N=0 \tag{2}N→+∞lim​aN​=0(2)
∑N=1∞aN=∞(3)\sum_{N=1}^\infty a_N=\infty \tag{3}N=1∑∞​aN​=∞(3)
∑N=1∞aN2<∞(4)\sum_{N=1}^\infty a_N^2<\infty \tag{4}N=1∑∞​aN2​<∞(4)

3.1 生成高斯分布的数据

import numpy as np
import matplotlib.pyplot as plt
# 生成高斯分布的数据
sample = 1.5*np.random.randn(5000000)+3

3.2 Robbins-Monro算法

# Robbins Monro 算法
sum_s = 0
hist = []
N = 1
for s in sample:
    sum_s += s
    theta_n = sum_s/N+1/N*(s-sum_s/N)
    N += 1
    hist.append(theta_n)

3.3 绘制参数的收敛曲线

# <center>RSSI模型参数动态修正的协作定位算法</center>
- RSSI测距模型的修正
  - 基于高斯滤波的RSSI测距优化
    - 可以有效降低短期干扰对RSSI测量值的影响，提高测距的抗干扰性和稳定性
  - 测距模型参数的修正
- 协作定位算法
  - 锚节点的选取：计算共线度，选出几组符合条件的锚节点（每组3个，尽量接近正三角形）
  - 基于加权的三边定位
    - 每组都计算出一个位置坐标
    - 越接近等边三角形，权重越高
    - 距离越近，权重越高
  - 协作定位的实现
    - 将已定位的未知节点假定为锚节点，将一个锚节点假定为未知节点，对其进行定位，最后计算误差
    - 选择误差偏小的未知节点升级为锚节点
